# Clear Floats

Clear floated child elements. 

## Installation

```
fonzie install clear-floats
```

## Usage

You have access to mixins and classes:

```scss
ul {
  @include clear-floats;
}
```

```html
<ul class="clear-floats">
  <li>Floated Item</li>
</ul>
```

You can also use the shorter `cf` class and mixin.
